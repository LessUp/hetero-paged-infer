## Overview

Hetero-Paged-Infer is a high-performance inference engine for Large Language Models (LLMs) that leverages CPU-GPU heterogeneous computing. It implements:

- **PagedAttention** for memory-efficient KV Cache management
- **Continuous Batching** for optimal throughput
- **Modular Architecture** with trait-based abstractions
- **Production-Ready** error handling and monitoring
